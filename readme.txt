This gradle build requires Java 11 jdk

In order to run, use the gradle run command and the main panel will show the buttons for questions 1-4

Note: All 4 questions are contained in one runnable build and seperate stages

Q1: press the "question 1" button to generate 3 random cards

Q2: Fill in the fields and press the calculate button

Q3: initally click and drag each circle to get each angle value aligned with the moveable circles

Q4: insert a file location starting with "C:\..." and press "view" to get the graph 
showing the frequency of letters in the file
